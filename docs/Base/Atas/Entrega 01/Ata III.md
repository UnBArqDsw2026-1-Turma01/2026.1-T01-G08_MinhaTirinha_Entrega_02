# Ata de Reunião 03 — Projeto de Software

* **Data:** 04 de abril de 2026.
* **Duração:** Aproximadamente 60 minutos.
* **Participantes:** Ana Carolina, Davi, Guilherme Flyan, João Marcelo, Maria Samara, Marjorie Mitzi, Pedro Henrique e Raissa.

## 1. Objetivo
A reunião teve como objetivo principal a definição dos requisitos iniciais do sistema, com base nas discussões anteriores e no direcionamento já estabelecido para o projeto de criação de tirinhas.

## 2. Discussões

### 2.1 Levantamento de requisitos
O grupo trabalhou na identificação e definição dos requisitos para consolidar as principais funcionalidades da aplicação. Os requisitos funcionais resumidos incluem:
* Criação de tirinhas.
* Edição de elementos visuais como personagens, cenários e textos.
* Salvamento e gerenciamento das tirinhas criadas.
* Compartilhamento das tirinhas.

### 2.2 Requisitos não funcionais
Foram definidos critérios de qualidade para a aplicação:
* Usabilidade e facilidade de uso.
* Interface intuitiva.
* Desempenho adequado.
* Compatibilidade com diferentes dispositivos.

### 2.3 Refinamento das ideias
Os requisitos foram discutidos de forma colaborativa para alinhar o entendimento entre os membros e garantir consistência com o escopo.

### 2.4 Relação com etapas anteriores
O grupo reconheceu que a definição dos requisitos está diretamente ligada às atividades anteriores, como o brainstorming e a definição do público-alvo.

## 3. Encaminhamentos
* Consolidar os requisitos definidos.
* Documentar os requisitos no GitHub Pages.
* Dar continuidade às atividades no Trello.
* Iniciar a prototipação com base nos requisitos.
* Agendar nova reunião.

## 4. Considerações finais
A reunião foi fundamental para consolidar o escopo inicial do sistema por meio da definição dos requisitos, permitindo o avanço para etapas mais concretas do desenvolvimento.
